ShortURL help page

Welcome, ShortUrl is REST web service, which acts as url redirect service.Service is packed as jar file and can be run as java -jar packageName. Currently when deployed service is listening on localhost:8080/ShortURL/

 

Account:

To use service, user first needs to create account by sending POST request containing username as String to /ShortURL/account.

Example:  {"username": "bib" }

Service returns username and password.

 

Register URL:

User sets basic authentication, and provides username and password. To register new url user sends POST request containing url and redirect type to /ShortURL/register. Redirect type is optional, if not provided it is set to 302 by default

Example:  { "initialUrl": "http://www.ABC.com", "redirectType": 301 }

Service returns short url and redirect type

 

Statistics:

To get statistics, just go to ShortURL/statistics, and service will provide list of all url's, together with count, to see how many times this url is accessed 

 

Redirect:

User is redirected to intial url by accessing ShortURL/{short} where short is short url which service provided when registering new url.
